# Segunda-Atividade-Pr-tica---Avaliativa-AP2-
Segunda Atividade Prática - Avaliativa  AP2 


Atividade:

Título: Cálculo de IMC com Stateful Widget

Descrição:

Nesta atividade, você irá desenvolver uma aplicação em Flutter que calcula o Índice de Massa Corporal (IMC) de uma pessoa. A aplicação deve seguir os seguintes passos:

1. Reproduza o protótipo fornecido:
image.png

2. Desenvolva a aplicação utilizando Stateful Widgets para gerenciar o estado da tela, permitindo que os valores dos campos de entrada e o resultado do cálculo sejam atualizados dinamicamente.

3. Implemente a lógica para calcular o IMC com base no peso e altura fornecidos pelo usuário. A fórmula para calcular o IMC é: IMC = peso / (altura * altura), onde o peso é em quilogramas e a altura em metros.

4. Apresente o resultado do cálculo do IMC de forma clara e legível na tela da aplicação.

5. Garanta que a aplicação previna erros, como divisão por zero ou entrada de valores inválidos nos campos de peso e altura.

Critérios de Avaliação:

- Resolução do Problema: A aplicação deve ser capaz de calcular corretamente o IMC com base nos valores fornecidos pelo usuário.
- Reprodução do Protótipo: A interface da aplicação deve ser semelhante ao protótipo fornecido, seguindo o design e layout especificados.
- Prevenção de Erros: A aplicação deve tratar possíveis erros de entrada do usuário, como valores inválidos ou divisão por zero.

Entregáveis:

- Aplicação no GitHub: Faça o upload do código-fonte da aplicação para o GitHub, organizado em um repositório adequado.
- Vídeo Explicativo: Grave um vídeo demonstrando e explicando o funcionamento da aplicação, destacando os principais aspectos do código e sua implementação.

Ao concluir a atividade, lembre-se de compartilhar o link do repositório do GitHub e o vídeo explicativo para avaliação e feedback.

Bons desenvolvimentos!
